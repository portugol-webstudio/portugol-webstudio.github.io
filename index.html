Feito para alunos

O Webstudio foi construído para ser o mais próximo do Portugol Studio, a fim de contemplar todos os usuários.

História

O Portugol Webstudio nasceu como uma adaptação do Portugol Studio, e a necessidade de torná-lo acessível em qualquer lugar. Como várias línguas de programação já possuem compiladores para navegadores, porquê não fazer um para o Portugol?
Foi então que @guiscaranse e @dgadelha junto com outros colaboradores construiram este projeto com o auxílio e poder do NodeJS.
